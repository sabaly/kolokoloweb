Le stockage des informtaions dans une base de données permets une meilleurs gestion du portfolio. Ce dernier est un ensemble de réalisations avec certains attributs : catégories, client, date de lancement du projet, lien, titre et une description.

Pour éviter de rajouter, à chaque nouveau projet, du code  dans les fichiers sources, la page `admin` permettra d'éviter celà en enregistrant le projet directement dans un base de données avec toute sa documentation.

Il faut protéger gérer l'accès à cet page.
